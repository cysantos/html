    Lista não Ordenada: ul

    Voltar ao início

    Em HTML para listarmos uma lista não ordenada podemos utilizar a tag "ul", porém assim como a tag anterior, essa só funciona se combinada com a tag "li". Conforme exemplo abaixo:

    • Item 3.3
